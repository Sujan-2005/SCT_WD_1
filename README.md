# Landing Page

A simple, responsive landing page built with HTML, CSS, and JavaScript.

## Files
- **index.html**: Main HTML structure.
- **styles.css**: Styles for layout, typography, and responsiveness.
- **script.js**: Interactive JavaScript functionality.

## How to Run
1. Open `index.html` in any modern web browser.
2. No build tools or servers required—runs directly!

## Features
- Clean, modern design.
- Responsive layout for mobile and desktop.
- Smooth interactions (explore script.js for details).

## Preview
Open `index.html` to view the landing page live.
